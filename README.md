# Bordplade

**Bordplade** aims to provide a set of tools for virtual tabletop and RPG sessions.

The repository is intended to be monolithic, in that it may contain both backend and frontend parts.

## Repository structure

~~~
/backend/
    + src/...
    + tests/...
/frontend/
    + node_modules/...
    + src/...
    - package.json
/deploy/
    - docker-compose.yml
    - Dockerfile
~~~
